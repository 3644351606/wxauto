适用于PC微信3.9版本

因有人发邮件说需要获取【`微信支付`】中的内容，则在原本的基础上稍作改动

```python
>>> from wxauto import *

# 获取当前微信客户端
>>> wx = WeChat()

# 首先检查版本信息是否正确
>>> print(UPDATE)  # 查看更新日期
2023-02-25
>>> print(VERSION) # 查看适用微信版本
3.9.0.28

# 输出当前聊天窗口聊天消息
>>> msgs = wx.GetAllMessage
>>> for msg in msgs:
...    print('%s : %s'%(msg[0], msg[1]))
...    print(msg[2], end='\n\n')    # 这里将返回的消息第三个元素改为了消息详情内容，如有需要可在这里获取。
```
